#迈动互联代码平台使用

网址：http://gitlab.maxrocky.com

##开发部分
添加公钥
登录网站进入个人设置>>SSH Keys>>ADD SSH KEY

获取公钥
公钥文件位置:  
linux/mac: ~/.ssh/id_rsa.pub  
windows: C:\Users\当前用户\.ssh    


克隆项目  
```
git clone git@gitlab.maxrocky.com:php/testproject.git
```
