利用satis配置私有包

Satis 是一个静态的 composer 代码库生成器, 可以用来处理私有资源包。

install

配置

新建配置文件 satis.json，设置公共库和私有库

完成配置通过以下命令生产项目 php bin/satis build <configuration file> <build dir>.

设置一个虚拟目录指向 web/ 目录, 假设绑定的域名是http://satis.yoursite.org。访问这个域名可以看到已经生成的包列表
